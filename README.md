# Handwritten Digit Classification Model with Perceptron, ANN and CNN

This notebook demonstrates handwritten digit classification on the **MNIST** dataset using three different neural network architectures implemented with **TensorFlow/Keras**.

## Models Implemented

- Perceptron (Single-Layer Neural Network)
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)

## Workflow

- Load the MNIST dataset
- Normalize image pixel values
- Prepare data for ANN and CNN models
- One-hot encode target labels
- Build and train each model
- Evaluate model performance
- Visualize predictions on test images
- Compare the learning capability of different architectures

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Dataset

- **MNIST Handwritten Digits**
- 70,000 grayscale images (28 × 28 pixels)
- 10 output classes (Digits 0–9)
